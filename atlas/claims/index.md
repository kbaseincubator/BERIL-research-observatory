# Claims

## What This Section Is For

Claims are compact statements that can be reused without rereading every project report. Each claim should preserve evidence, provenance, confidence, and caveats.

## How To Use Claims

Use a claim when a topic or proposal needs a stable premise. If the claim is low confidence or caveat-heavy, treat it as a hypothesis seed rather than a settled fact.

## Maintenance Rule

Claims should be revised when new projects change the evidence balance, add conflicting results, or improve the underlying data joins.

# Opportunities

Opportunities are the Atlas layer for action. They translate synthesis, tensions, reusable products, and data gaps into concrete analyses or experiments that a human or agent can assess without rereading every source project.

## How To Read This Section

Start with high-impact, high-readiness opportunities when looking for the next tractable analysis. Start with blocked opportunities when deciding which missing data or review would unlock multiple Atlas topics. Each page records source projects, linked tensions, reusable products, target outputs, and review routes.

## Maintenance Rule

Opportunity pages are not project proposals. They are pre-proposal decision objects: enough structure to compare value and feasibility, but still small enough to revise as the Atlas changes.

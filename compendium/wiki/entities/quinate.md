# Entity: Quinate

## Introduction

Quinate is the named carbon-fitness exception in the current wiki. It appears in deletion phenotyping as the main discrete exception to the continuous ADP1 carbon-fitness landscape and in explorer evidence as an outlier condition [stmt:adp1-deletion-quinate-module-finding; adp1_deletion_phenotypes] [stmt:adp1-explorer-condition-fitness-finding; acinetobacter_adp1_explorer].

## Synthesis

The entity matters because quinate marks where general carbon-fitness synthesis should become condition-specific. It should be used to test whether a local module explains a distinct fitness pattern rather than forcing all carbon-source behavior into one continuous trend [stmt:adp1-deletion-quinate-module-finding; adp1_deletion_phenotypes].

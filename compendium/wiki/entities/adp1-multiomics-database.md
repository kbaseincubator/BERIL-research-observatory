# Entity: Adp1 Multiomics Database

## Introduction

The ADP1 multiomics database is the collection entity for the explorer's integrated data product. It represents the integration of six molecular and phenotype data modalities for Acinetobacter baylyi ADP1 [stmt:adp1-explorer-multiomics-finding; acinetobacter_adp1_explorer].

## Synthesis

This entity is narrow but important: it is the concrete data product behind the explorer's role as a BERDL bridge and downstream synthesis surface. The database gives other pages a shared place to connect molecular and phenotype evidence instead of treating each modality as a separate project artifact [stmt:adp1-explorer-multiomics-finding; acinetobacter_adp1_explorer].

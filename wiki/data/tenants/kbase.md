# KBase Tenant and Foundational Genome Data

## Role In The Observatory

KBase collections provide the genome-scale substrate for most projects: genome identifiers, pangenome families, gene annotations, pathway predictions, phenotype links, and biochemical reference mappings.

## Main Collections

- [Pangenome Collection](/wiki/data/collections/pangenome)
- KBase Genomes
- ModelSEED Biochemistry
- Phenotype Collection

## Agent Use

Use this tenant when a question starts with genomes, genes, pangenome structure, functional annotation, or metabolic pathways. Pair it with Fitness Browser when a claim needs experimental validation.

# Wiki Roadmap

## Phase 0

Static agent-authored markdown atlas, UI browsing, frontmatter graph, and lint checks.

## Phase 1

Drafting workflow that scans project outputs and proposes wiki diffs for review.

## Phase 2

Structured claim/evidence model, derived data product registry, and source coverage dashboards.

## Phase 3

Agent maintenance loop with stale-page detection, review routing, and proposed updates.

## Phase 4

Retrieval layer, backlinks, topic graph, and agent-facing APIs for planning and synthesis.

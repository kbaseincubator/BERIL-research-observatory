# Data Dictionary

All CSV files are generated by analysis notebooks and gitignored (too large for git). They are archived to BERDL MinIO after submission.

| File | Rows | Produced by | Description |
|------|------|-------------|-------------|
| `amr_census.csv` | 83,008 | NB01 | One row per AMR cluster: gene_cluster_id, amr_gene, amr_product, method, identity, coverage, conservation_class, mechanism, taxonomy (phylum→genus), annotation depth (has_bakta_product, has_eggnog, has_pfam, n_sources) |
| `amr_species_summary.csv` | 14,723 | NB01 | One row per AMR-carrying species: n_amr, n_amr_genes, n_core/aux/sing_amr, n_mechanisms, taxonomy, pangenome stats (no_genomes, no_gene_clusters, openness) |
| `amr_conservation_by_gene.csv` | 1,939 | NB02 | One row per AMR gene family: total count, n_core/aux/sing, n_species, n_phyla, mean_identity, pct_core |
| `amr_conservation_by_species.csv` | 14,723 | NB02 | Per-species conservation comparison: amr_core_frac vs species_core_frac, core_enrichment (difference) |
| `amr_cog_enrichment.csv` | 21 | NB04 | One row per COG category: count_amr, count_baseline, amr_frac, base_frac, enrichment ratio |
| `amr_by_environment.csv` | 7,838 | NB05 | Species with environment classification: primary_env (Human/Clinical, Soil, Aquatic, etc.) + all AMR/pangenome columns |
| `amr_phylum_summary.csv` | 94 | NB01 | Phylum-level aggregation: n_species_with_amr, total_amr, mean/median_amr, mean_amr_density, mean_pct_core, total_species, pct_species_with_amr |
| `amr_by_family.csv` | 1,209 | NB03 | Family-level aggregation: n_species, total_amr, mean/median/max_amr, mean_openness, mean_genomes |
| `amr_by_genus.csv` | 4,257 | NB03 | Genus-level aggregation: same columns as family |
| `amr_fitness.csv` | 29,386 | NB06 | Fitness measurements for AMR genes: orgId, locusId, fitness, t_score, amr_gene, mechanism, conservation_class |

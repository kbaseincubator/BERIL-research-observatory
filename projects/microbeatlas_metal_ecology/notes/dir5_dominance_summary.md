# Direction 5: Community Dominance Near Mines — 2026-04-04

## Hypothesis
If metal stress causes **competitive exclusion** (a few metal-tolerant taxa monopolize reads),
then mine-proximal desert communities should show low richness AND high reads-per-OTU (dominance).
If metal stress causes **general inhibition**, dominance should be unchanged near mines.

Metric: `reads_per_OTU = total_counts / n_genes_by_counts` (Berger-Parker dominance proxy)

## Desert Results (primary environment)

- n = 1355 samples with mine distance
- Near (<5 km):  n=410, median reads/OTU = 169.3
- Mid (5–25 km): n=945, median reads/OTU = 62.6
- Far (>25 km):  n=0, median reads/OTU = nan

Near vs Mid FC (near/mid) = 2.71 → dominance INCREASES near mine (competitive exclusion (fewer taxa monopolize reads))
Near vs Mid MW p = 2.38e-31
Near vs Far FC (near/far) = nan, p = n/a

Spearman(dist_km, reads_per_OTU): ρ = -0.209, p = 8.05e-15
Spearman(richness, reads_per_OTU): ρ = -0.610, p = 7.95e-139

OLS reads_per_OTU ~ log_dist: β = -54.2, p = 8.38e-05, R² = 0.011
OLS reads_per_OTU ~ log_dist + log_depth: β(dist) = -53.7, p = 5.82e-05, R² = 0.072
OLS log(reads_per_OTU) ~ log_dist + log_depth: β(dist) = -0.378, p = 1.08e-17, R² = 0.095

For reference:
OLS otu_richness ~ log_dist + log_depth: β(dist) = +133.1, p = 1.38e-09, R² = 0.273

## Interpretation

- If dominance FC > 1 (near > mid) and p < 0.05: evidence for **competitive exclusion** near mines
- If dominance FC ≈ 1 or ns: evidence for **general inhibition** (uniform suppression)
- Combined with richness suppression: characterizes the mechanistic mode of metal toxicity

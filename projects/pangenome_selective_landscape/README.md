# The Selective Landscape of Pangenome Architecture

## Research Question
Across 48 Fitness Browser organisms with pangenome links, does the **type** of lab condition under which a gene is important (metal stress, carbon source, antibiotic, nitrogen, osmotic) predict whether that gene resides in the core or accessory genome?

## Status
Exploration — research plan not yet written.

## Overview
Previous BERIL projects established that fitness importance predicts pangenome conservation (16pp gradient from essential to neutral) and that core genes are paradoxically costly. None broke this down by **condition type**. This project asks which environmental pressures actually shape pangenome architecture — testing whether metal-stress genes, carbon-source genes, and antibiotic-resistance genes show different core/accessory distributions across 48 organisms and 350 conditions. The condition-breadth hypothesis proposes a mechanism for core genome assembly: genes become core by being important under many conditions.

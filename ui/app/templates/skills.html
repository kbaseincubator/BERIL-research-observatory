How Skills Work

Skills are invoked by the AI co-scientist during research sessions using slash commands (e.g., /synthesize, /literature-review). Each skill reads from project files, queries BERDL data, and produces structured outputs like reports, references, or data quality checks.

Skills are defined as markdown specifications in .claude/skills/ and are version-controlled alongside the research artifacts they produce.
